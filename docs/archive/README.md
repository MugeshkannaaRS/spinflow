# Archive

Historical audit, planning, and release reports generated during SpinFlow's
development. These are kept for reference but are **not** active project
documentation — the current docs live at the repo root (`README.md`,
`AGENTS.md`, `BACKEND.md`, `DEPLOYMENT.md`, `TESTING.md`, `STAGING.md`).

Contents:

- Audit & readiness reports (`AUDIT_REPORT`, `*_PRODUCTION_READINESS_REPORT`,
  `SPINFLOW_ENTERPRISE_AUDIT`, `spinflow_qa_report`, `AUDIT_PACKAGE/`)
- Release-candidate notes (`RC-1_REPORT`, `RC-1.1_REPORT`, `TRACKS_AB_REPORT`)
- Architecture & planning docs (`SpinFlow_Architecture_*`,
  `SPINFLOW_MASTER_EXECUTION_PLAN`, `WAVE4_ENTERPRISE_BLUEPRINT`,
  `SpinFlow_Production_Module_Plan_v3`)
- Remediation / customization notes and the large session `CONTEXT.md`
- Requirements / schedule deliverables (`SpinFlow_PM_*.docx`, `.xlsx`)
